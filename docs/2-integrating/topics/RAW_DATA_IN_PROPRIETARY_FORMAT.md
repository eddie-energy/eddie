# Topic: Raw Data In Proprietary Format

The topic **raw-data-in-proprietary-format** delivers the data of each region in their original form. Some countries
like Austria deliver their data as XML while other countries such as Denmark or France deliver their data as JSON.
A documentation for each country original format can be found under the following links. The links were last checked
on 20th of Feburary 2024.

* [Austria](https://www.ebutilities.at/schemas/149) (see section links and documents)
* [Denmark](https://api.eloverblik.dk/CustomerApi/index.html) (see meterdata/gettimeseries)
* [France](https://datahub-enedis.fr/services-api/data-connect/documentation/) (see daily consumtion and consumption)
  load curve)
* Spain (Currently no official documentation available)
