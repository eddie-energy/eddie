# Integrating

EDDIE can integrate with other systems using several protocols in the form of [outbound Connectors](../1-running/outbound-connectors/outbound-connectors.md).
Permission request status changes, the requested data, and terminations are described in [messages](messages/messages.md).

EDDIE creates permission requests based on data needs.
- [Data needs](./data-needs.md) are a data definition language that defines the format of permission requests and the data that is requested.
- [Permission requests](./permission-requests.md) track the state of a final customer's consent and the data provider's ability to share requested data.
